Conway's Game of Life

Developed the backend logic of John Conway's Game of Life using java. Receives text input from the user to create the 2D array board, methods that identify alive cells and their neighbors, computing the grid's new generations following the game’s rules, and count the number of communities using weighted-quick-union

Huffman Coding (Data Compression)

Implemented a text file compression program using Huffman’s algorithm. Created and edited binary search trees of nodes resembling all the characters in an input file, in order of the frequency of their appearance, and created and manipulated queues in order to create the Huffman Tree. Traverse our binary search tree to create an array of “encodings” representing characters used in the text file to encode and decode the file.

Kindergarten Class Simulator

Created a kindergarten simulation that has students in line (a singularly linked list), and has the options to seat them in a 2d array based on seating availability, put them into a musical chairs circular linked list, play a game of musical chairs, put them back into their seats starting from the winner, options to delete and add students from any of the data structures mentioned.
